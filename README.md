# PrivEsc

PrivEsc is a practical reference for Linux and Windows privilege escalation workflows, commands, and exploitation patterns.

## Live Project

- Website: https://medtrigui.github.io/privilege-escalation/

## Guides

- Linux Guide: https://medtrigui.github.io/privilege-escalation/docs/linux-privilege-escalation/
- Windows Guide: https://medtrigui.github.io/privilege-escalation/docs/windows-privilege-escalation/

## Main Tools & Resources

- GTFOBins: https://gtfobins.org/
- Escabin: https://medtrigui.github.io/escabin/
- SearchSploit / Exploit-DB: https://www.exploit-db.com/
- LinPEAS / WinPEAS (enumeration helpers)
